Privacy Policy

Subsly

Last updated: May 2026

1. Overview

Subsly is an application available as a website and as a browser extension. The web version can be used both without logging in and while logged in to a Subsly account; the browser extension requires a logged-in session. This Privacy Policy explains what information the extension and web app collect, how it is used, where it is stored, and the choices available to users.

2. Information We Collect

Subsly does not collect personally identifiable information without the user's knowledge and explicit action.

a) User-provided data

When using the app or the extension, you may voluntarily enter data related to your subscriptions, such as:

  • Subscription names
  • Prices
  • Billing periods
  • Categories, notes or other metadata you enter

b) Technical data and permissions

The extension requests the minimal permissions required to operate (for example, Chrome storage APIs). Permissions are not used for tracking, advertising, or profiling.

3. How Data Is Used

  • Allow users to create and manage subscriptions
  • Improve usability and stability of the product

Data is not used for targeted advertising or commercial profiling.

4. Data Storage

  • In the browser's local storage (e.g., storage APIs) when using the web app without logging in.
  • In your Subsly account when you are logged in. In that case, data is stored on our servers or on third-party services acting as processors (hosting providers).

5. Data Sharing

Subsly does not sell, rent, or share user data with third parties for marketing purposes. We may share information with processors (e.g., hosting or authentication services) only as necessary to provide the service.

6. Remote Code

The extension does not execute remotely hosted code. All executable code is packaged within the extension bundle. This aligns with Chrome Web Store security policies.

7. Security

We implement reasonable technical and organizational measures to protect user data and prevent unauthorized access. However, no system can guarantee absolute security; please follow best practices (strong passwords, keep your browser updated, etc.).

8. User Control

You keep full control over your data. Available options:

  • Remove individual subscriptions via the interface
  • Clear data stored in the browser
  • Delete your Subsly account (if applicable) to remove associated server-side data
  • Uninstall the extension at any time

9. Chrome Web Store Publication

The extension will be published in the Chrome Web Store. Prior to publication we will declare the required permissions and comply with store policies. The extension will request only the minimum permissions necessary to operate.

10. Contact

If you have any questions about this Privacy Policy or wish to exercise your rights (access, rectification, deletion), please contact the development team via the project repository: https://github.com/Waxer59/Subsly.

Thank you for using Subsly.
